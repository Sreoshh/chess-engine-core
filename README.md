#  Chess Engine Core 

A simple (to me it is better than stockfish though </>) Java based chess engine built from scratch.

## Features

* Move generation
* Move validation
* Minimax AI with alpha-beta pruning
* Board evaluation

## Structure

* `core/` → basic models (Board, Piece, Position)
* `engine/` → AI logic (Search, Evaluator)
* `game/` → game flow & rules

## Usage

This is a backend engine so can be integrated into anything be it android or web apps. Tweak and make a better version of it ;))


## Future Improvements

* Checkmate & stalemate detection improvements
* Better evaluation heuristics
* GUI integration
